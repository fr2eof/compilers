# Учебный компилятор

Минимальный компилятор и интерпретатор на C#, демонстрирующий полный цикл обработки исходного кода: от лексического анализа до выполнения программы.

## Возможности

* Лексический разбор исходного текста
* Построение абстрактного синтаксического дерева (AST)
* Проверка типов и выполнение базовых семантических проверок
* Оптимизация AST:
    * свертка констант
    * удаление недостижимого и неиспользуемого кода
* Генерация промежуточного представления LLMR
* Выполнение программы через интерпретатор

## Структура проекта

* `Lexer.cs`, `Token.cs`, `TokenType.cs` - лексический анализ и токенизация
* `Parser.cs`, `Statement.cs`, `Expression.cs` - построение AST
* `TypeChecker.cs`, `UnusedVariableChecker.cs`, `UninitializedVariableChecker.cs` - семантический анализ
* `ConstantFolder.cs`, `DeadCodeEliminator.cs` - оптимизации
* `LlmrTranslator.cs`, `LlmrInstruction.cs` - генерация LLMR
* `Interpreter.cs`, `RuntimeEnvironment.cs` - интерпретация и выполнение
* `Program.cs` - объединение всех этапов компиляции в единый пайплайн

## Запуск

Откройте решение `compilers.sln` и запустите проект `Lexer`.

## Поддержка массивов

Компилятор поддерживает создание массивов в формате `[1, 2, 3]`, а также чтение и изменение элементов по индексу, например: `arr[0] = 5;`.

* На этапе компиляции `TypeChecker` контролирует типы элементов массива и не допускает смешивания разных типов данных (например, строк и чисел в одном массиве).
* Во время выполнения `Interpreter` проверяет корректность индексов и предотвращает выход за границы массива (Index Out Of Bounds).
